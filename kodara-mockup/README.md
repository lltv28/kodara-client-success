# Kodara Mockup

This is a React + Tailwind CSS implementation of the Kodara UI mockup.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open the link provided in the terminal (usually http://localhost:5173).

## Project Structure

-   `src/components/Sidebar.tsx`: The left sidebar navigation.
-   `src/components/MainContent.tsx`: The main chat and input area.
-   `src/App.tsx`: Main layout.
